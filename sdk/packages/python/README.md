# PromptChain Python SDK

Python SDK for the PromptChain protocol on Solana.

```python
from promptchain import PromptChainClient

client = PromptChainClient("https://api.devnet.solana.com")
```

## Features

- Publish, list, fetch, and fork prompts
- Manage licenses and transfers
- Curation and rating
- Token economics (staking, minting)
- DAO governance (proposals, voting)
